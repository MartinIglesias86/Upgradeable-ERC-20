# ✨ Upgradeable ERC20

This project is an upgradeable ERC20 token.
You can find 3 versions of the token contract, a forwarder contract and an EIP712 contract.
The project also includes the tests to check the contracts (test folder)
 
## 🔧 Characteristics and commands

### 👨🏻‍💻 Install
To run the project install all the dependencies using the following command:

`npm i`
